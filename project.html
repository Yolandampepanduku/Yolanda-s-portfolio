Team-Alpha-Project

This project is about combining local news with job opportunities.

Project Overview

Our proposed project is Community Connect which is a mobile and web-based platform that combines local news with job opportunities. It will include features such as Business and job seeker profiles,Job posting and searching and Location filters (5km range).

Project Features

  • Feature 1: Local news feed
  • Feature 2: One-click job applications
  • Feature 3: Basic CV builder
